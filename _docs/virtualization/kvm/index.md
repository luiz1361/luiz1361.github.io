title: KVM - Migrate VM Other host
category: Virtualization
---

**On source host:**
```
virsh shutdown vm
virsh dumpxml vm > /tmp/vm.xml
scp /tmp/vm.xml kvm02:/tmp/vm.xml
scp /var/lib/libvirt/images/vm.qcow2 kvm02:/var/lib/libvirt/images/vm.qcow2
virsh undefine vm
rm /var/lib/libvirt/images/vm.qcow2
```

**On target host:**
```
virsh define /tmp/vm.xml
virsh start vm
```

**Install headless:**
```
apt-get install qemu-kvm libvirt-bin virtinst bridge-utils cpu-checker
```

**Check install and virtualization acceleration:**
```
kvm-ok
```

**List VMs:**
```
virsh list
```

**Start VM:**
```
virsh start openbsd
```

**Suspend VM:**
```
virsh suspend openbsd
```

**Soft reboot:**
```
virsh reboot openbsd
```

**Hard reset:**
```
virsh reset openbsd
```

**Shutdown:**
```
virsh shutdown openbsd
```

**Remove VM:**
```
virsh undefine openbsd
virsh destroy openbsd
```
