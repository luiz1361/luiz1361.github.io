**Installing TFTPD-HPA**
```
apt-get install tftpd-hpa
```

**Working folder:**
```
/srv/tftpd
```

**Check default file:**
```
cat /etc/default/tftpd-hba
```

**Start daemon:**
```
/etc/init.d/tftpd-hba start
```
